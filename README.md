# 🚀 SKY ROCKET: JAVA FORCE

![Java](https://img.shields.io/badge/Java-17-orange)
![Platform](https://img.shields.io/badge/Platform-Desktop-blue)
![IDE](https://img.shields.io/badge/IDE-IntelliJ_IDEA-red)
![License](https://img.shields.io/badge/License-Educational-green)

A 2D space shooter game developed using **Java** as a Data Structures course project at **Astra Polytechnic**.

The game demonstrates Object-Oriented Programming (OOP) principles and practical implementation of various data structures through an interactive desktop game.

---

# 🎮 Features

- 🚀 Player spaceship movement
- 🔫 Shooting system
- 👾 Random enemy rockets
- ❤️ Health System
- 💚 Health Pack pickup
- 💥 Explosion animation
- 🎵 Background music & sound effects
- 🎬 Intro, Victory, and Game Over videos
- 🏆 Leaderboard system
- 🌌 Dynamic game backgrounds
- 📈 Multiple difficulty levels
- ⚡ Smooth game loop

---

# 📚 Concepts Implemented

- Object-Oriented Programming (OOP)
- Java Swing GUI
- Event Handling
- Collision Detection
- Game Loop
- Animation
- Audio Processing
- Video Playback (VLCJ)

# ▶️ How to Run

## Requirements

- Java JDK 17+
- IntelliJ IDEA
- VLC Media Player (required by VLCJ)

Clone repository

```bash
git clone https://github.com/gumelarakhirul/SkyRocket-Java-Force.git
```

Open using IntelliJ IDEA

Run

```
Main.java
```
